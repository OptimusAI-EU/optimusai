## Key UI Features Explained

### 1. Summary Statistics Cards (BOTH PANELS)

**Functionality:**
- Display aggregated counts by category
- **Clickable** - click any card to filter the table below
- Card highlights (red border) when filter is active
- Shows "Click to filter" hint on hover

**Orders Panel:**
- Product Types: Robots | ODIN Subscriptions
- Order Status: Cart | Awaiting Shipping | Shipped | Delivered | Cancelled

**Subscriptions Panel:**
- Plan Types: All combinations (Starter Monthly, Starter Annual, Professional Monthly, etc.)
- Status: Cart | Active | Awaiting Renewal | Cancelled

### 2. Phone Column (BOTH PANELS)

- Displays user's phone number
- Shows "N/A" if not provided
- Searchable - filter by phone number
- Visible in expanded detail view

### 3. Product Type Badge (Orders Only)

**Color coding for easy identification:**

Orders:
- Cart: Blue
- Awaiting Shipping: Yellow  
- Shipped: Purple
- Delivered: Green
- Cancelled: Red

Subscriptions:
- Cart: Blue
- Active: Green
- Awaiting Renewal: Yellow
- Cancelled: Red

### 6. Expandable Rows

- Click "View" to expand, "Hide" to collapse
- Shows detailed information
- Allows status updates (Orders)
- Shows full customer contact info and addresses
- Displays subscription dates with full timestamps

---

## Responsive Design

All panels are responsive:
- **Mobile:** Single column layout
- **Tablet:** 2-3 column grid for summary cards
- **Desktop:** Full 4-5 column grid
- **Tables:** Horizontal scroll on small screens

---

## Interactive Elements

1. **Summary Cards:** Click to filter table
2. **Search Box:** Type to search across multiple fields
3. **Filter Dropdowns:** Select specific statuses or types
4. **Expand/Hide Links:** Toggle detailed view
5. **Status Dropdowns (Orders):** Change order status with confirmation
6. **Cancel Button (Subscriptions):** Cancel subscription with confirmation
7. **Confirmation Modals:** Prevent accidental changes

---

## Search Capabilities

**AdminOrders Search includes:**
- Order number
- Customer email
- Customer name (first + last)
- Customer phone number

**AdminSubscriptions Search includes:**
- Customer email
- Customer name (first + last)
- Customer phone number
- Plan name

---

## Future Enhancement Opportunities

1. **Bulk Actions:** Select multiple orders/subscriptions and perform actions
2. **Export:** Export filtered data to CSV/Excel
3. **Advanced Reports:** Revenue by product type, churn analysis, etc.
4. **Automation:** One-click bulk status updates
5. **Notifications:** Email customers when status changes
6. **API Integration:** Sync with payment processors, shipping providers
7. **Custom Fields:** Add custom metadata to orders/subscriptions
8. **Audit Trail:** Track all status changes with timestamps and user info
